5. *Самостоятельно разобраться с атрибутами тега script (async и defer)

Ответ:
Атрибут defer откладывает выполнение скрипта до тех пор, пока вся страница не будет загружена полностью.
Конструкция async/await используется для решения асинхронных задач и не блокирует главный поток.
